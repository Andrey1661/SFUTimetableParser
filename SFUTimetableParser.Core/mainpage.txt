 * \section intro О проекте
 * \subsection Введение
 * Данный проект является прикладным средством для работы с расписанием СФУ.\n
 * Основная задача проекта - парсинг страницы официального сайта СФУ, содержашей расписание учебных групп.\n
 * \subsection Задачи
 * В задачи данной подсистемы входит следующее:\n
 * 1) Считывание информации о существующих учебных группах и институтах;\n
 * 2) Считывание информации о расписании всех учебных групп либо заданной;\n
 * 3) Преобразование считанной информации во внутренний формат для дальнейшей работы (см. SFUTimetableParser.Core.Entities);\n
 * 
 * \subsection Структура
 * Проект состоит из двух частей:\n
 * Ядро – содержит описания интерфейсов, моделей данных, исключений, создающих основу приложения (SFUTimetableParser.Core);\n
 * Базовая реализация – содержит реализацию интерфейсов ядра, является основной функциональной частью системы (SFUTimetableParser.Implementation);\n
 */
